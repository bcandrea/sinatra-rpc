# Sinatra::Rpc

A simple Sinatra extension providing the functionality of an RPC server. 

## Installation

Add this line to your application's Gemfile:

    gem 'sinatra-rpc'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sinatra-rpc
